# Airsoft events app

## project brief
Airsoft event planner app. users can browse posted airsoft events, register to games. game organizers can build rosters for the game, assign players into teams. 
users can apply to become organizers, uppon acceptance, are allowed to advertise events. the app can send reminders, updates on games that they have signed up.
users also can discuss events by leaving comments, replies. 

## tech stack
typescript, react router 7, bun, pino, hono, better auth, 

Built with ❤️ using React Router.
